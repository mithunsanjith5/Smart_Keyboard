Android System-Wide Keyboard (Java)

This project is an Android IME (Input Method Editor), which is how system-wide keyboards work on Android.

Project path:
/home/mithun-sanjith/Documents/workspace/college/pcl/android-system-keyboard

How to build and install:
1. Open this folder in Android Studio.
2. Let Android Studio sync Gradle.
3. Connect your Android phone with USB debugging enabled.
4. Press Run in Android Studio and install the app.

How to enable keyboard on phone:
1. Open the installed app: Android System Keyboard.
2. Tap Open Keyboard Settings.
3. Enable Android System Keyboard in the keyboard list.
4. Tap Choose Keyboard and select Android System Keyboard.

Notes:
- Android requires keyboard apps to run as IME services, not regular background key hooks.
- This is a minimal keyboard with letters, space, enter, backspace, comma, and period.
